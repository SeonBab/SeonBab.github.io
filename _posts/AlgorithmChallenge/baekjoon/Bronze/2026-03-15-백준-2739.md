# 구구단

## 문제 링크

[구구단](https://www.acmicpc.net/problem/2739){: target="_blank"}

## 분석

입력받은 값에 대한 구구단을 출력하면 됩니다.

반복문을 사용해 풀이할 수 있습니다.

## 풀이

```cpp
#include <iostream>

using namespace std;

int main()
{
	int N;

	cin >> N;

	for (int i = 1; i <= 9; i++)
	{
		cout << N << " * "<< i << " = " << N * i << endl;
	}

	return 0;
}
```
