# A+B-3

## 문제 링크

[A+B-3](https://www.acmicpc.net/problem/10950){: target="_blank"}

## 분석

첫 번째 줄에 테스트 케이스의 개수가 주어집니다.  
이후 해당 개수만큼의 줄에 걸쳐 정수 A와 B가 주어지며, 각 줄마다 해당 정수들의 합을 출력하면 됩니다.

개수를 입력받고, 해당 개수만큼 반복문으로 반복하며 값을 더하고 출력하면 되는 문제입니다.

## 풀이

```cpp
#include <iostream>

using namespace std;

int main(void)
{
	int T, a, b;

	cin >> T;

	for (int t = 0; t < T; t++)
	{
		cin >> a >> b;
		cout << a + b << endl;
	}

}
```

## 성능 요약

시간 복잡도는 $O(T)$입니다.

- 테스트 케이스의 개수만큼 반복하는 반복문 $O(T)$

공간 복잡도는 고정된 크기의 상수 공간을 사용하기 때문에 $O(1)$입니다.
