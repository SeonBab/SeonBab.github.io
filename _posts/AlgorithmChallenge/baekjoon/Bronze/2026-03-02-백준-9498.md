# 시험 성적

## 문제 링크

[시험 성적](https://www.acmicpc.net/problem/9498){: target="_blank"}

## 분석

정수가 하나 주어지며, 정수에 대한 값을 성적으로 변환해 출력하는 문제입니다.

점수의 범위와 그 범위에 대해 성적은 정해져 있습니다.

100점부터 10점씩 A부터 F까지 한 단계씩 줄어듭니다.  
즉, 100 ~ 90이 A라고 했을 때 89부터 80은 B가 됩니다.

해당 문제는 조건식으로 문제를 풀이할 수 있습니다.  
가장 간단하고 직관적인 방식은 if-else 문입니다.

## 풀이

```cpp
#include <iostream>

using namespace std;

int main(void)
{
	int a;

	cin >> a;

	if (a >= 90)
	{
		cout << "A";
	}
	else if (a >= 80)
	{
		cout << "B";
	}
	else if (a >= 70)
	{
		cout << "C";
	}
	else if (a >= 60)
	{
		cout << "D";
	}
	else
	{
		cout << "F";
	}

}
```
