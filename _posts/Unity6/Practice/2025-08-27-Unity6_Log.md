# 디버그 로그 출력하기

로그를 출력할 때는 `Debug` 클래스를 사용합니다.

## 일반 로그 출력

```C#
Debug.Log("일반 로그 메시지");
```

## 경고 로그 출력

```C#
Debug.LogWarning("경고 로그 메시지");
```

## 에러 로그 출력

```C#
Debug.LogError("에러 로그 메시지");
```

## 오브젝트 정보 추가

```C#
GameObject Player = GameObject.Find("Player");

Debug.Log("플레이어 위치:" + Player.transform.position, Player);
```
